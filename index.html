Code Print

Auto-generate markup examples for your styleguide.

Download ZIP

If you like to handcraft your styleguides, but don't like to handcraft the code examples then this script is for you. Heavily inspired by the excellent xrayhtml.

Usage

Include jquery and codeprint in your page:

If you want to render a code snippet of an element, then simply add the data-attribute data-codeprint. (Don't worry, codeprint will strip out this attribute from the markup example it generates).

If you want to render the markup for a group of elements then simply wrap in a div with the data attribute data-codeprint-group.
